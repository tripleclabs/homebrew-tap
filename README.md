# Homebrew Tap for Triple C Labs

Homebrew formulae for [Triple C Labs](https://github.com/tripleclabs) projects.

## Install

```bash
brew tap tripleclabs/tap
brew install nova
```

## Packages

| Formula | Description |
|---|---|
| [nova](https://github.com/tripleclabs/nova) | Next-generation local VM orchestrator — a modern replacement for Vagrant |

## Updating

```bash
brew update
brew upgrade nova
```

## Build from source (HEAD)

```bash
brew install --HEAD tripleclabs/tap/nova
```
